Project Site Plan

Site Name

Raspberry Pi Foundation Microcontrollers. This page is focused on some microcontrollers that are developed by the Raspberry Pi Foundation. A possible domain site name could be: rpi-mcus.org

Site Purpose

This site will try to summarize some information about microcontrollers developed by the Raspberry Pi Foundation, encouraging students and electrical engeniers to try them out and design some interesting products.

Scenarios

  • What projects I could work on with these microcontrollers?
  • What are some of the main characteristics of these microcontrollers?

Color Schema

The colors chosen will be simple: #000000 and #333333 for font color, #FFFFFF and #F5F6F9 for backgrounds and #53ce63 for buttons.
